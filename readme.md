# SecureOps: Website Locking Extension  

SecureOps is a Chrome extension designed to provide an additional layer of security for your browsing experience. It allows users to lock specific websites with a master password, ensuring sensitive data and login credentials are protected.  

---

## Features  

- **Website Locking**: Lock specific websites with a master password for added security.  
- **Persistent Protection**: Retain locked websites even after browser restarts.  
- **Master Password Management**: Manage access to locked websites using a secure master password.  
- **User-Friendly Interface**: Easily manage locked websites and passwords via a clean UI.  

---

## Installation  

1. Clone or download this repository.  
2. Open Chrome and navigate to `chrome://extensions/`.  
3. Enable **Developer Mode** in the top right corner.  
4. Click **Load unpacked** and select the project folder.  

---

## How It Works  

1. **Register**: Set up your master password upon installation.  
2. **Lock Websites**: Add website URLs to the locked list via the extension popup.  
3. **Unlock Websites**: Enter your master password to temporarily access locked websites.
